# 💡 IoT Smart Lighting System

An IoT-based smart lighting system developed using **ESP8266**, **LDR**, **MIT App Inventor**, and **Telegram Bot**. The system enables automatic and manual control of lighting while providing real-time notifications for enhanced convenience and energy efficiency.

---

## 📌 Features

- Automatic light control based on ambient light intensity
- Manual ON/OFF control using an Android application
- Wireless communication using ESP8266 Wi-Fi module
- Real-time Telegram notifications
- Energy-efficient lighting automation
- User-friendly mobile interface

---

## 🛠️ Hardware Components

- ESP8266 Wi-Fi Module
- LDR (Light Dependent Resistor)
- LED
- Breadboard
- Jumper Wires
- USB Cable
- Power Supply

---

## 💻 Software Used

- Arduino IDE
- Embedded C++ (Arduino)
- MIT App Inventor
- Telegram Bot API

---

## ⚙️ Working Principle

The LDR continuously monitors the surrounding light intensity. Based on the detected light level, the ESP8266 automatically controls the LED. Users can also manually operate the light using a mobile application built with MIT App Inventor. Telegram Bot integration provides real-time notifications and remote interaction over Wi-Fi.
